User Login Details

Track User Login Activity with IP and Timestamp

This module enables administrators to track which users have logged in to a newly created or cloned Odoo database. It logs the username, login time, and system IP address to provide full visibility and accountability. The menu for this feature will be available under General Settings >User & Companies >Login Details.

Usage:

After installing the module, navigate to General Settings >User & Companies >Login Details To view records. Each entry shows the user's name, login time, and IP address of the system used for login.

Technical Details:
